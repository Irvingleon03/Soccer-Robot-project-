# Soccer Robot Project 🤖⚽
This repository documents the mechanical design and development of the Soccer Robot Project for a competition.



## Technical Specifications
**Locomotion:** 3-Wheel Omnidirectional Drive for high-agility movement, the wheels are positioned 120 degrees apart.

**Brain:** Raspberry Pi 5 for high-level processing and vision.

**Power & Control:** 2 L298N Dual H-Bridge Motor Drivers for DC motor management, providing 4 independent channels for the 3-motor holonomic system and auxiliary actuators.

 **Design:** Fully modeled using SolidWorks.

## Repository Structure
 **Mechanical Design:** Includes `.SLDPRT` (parts) and `.SLDASM` (assemblies).
 
 **Software:** (Upcoming) Computer vision scripts using Python and OpenCV.
